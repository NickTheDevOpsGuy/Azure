# Azure Kubernetes Service (AKS) Lab

This repository contains labs and projects demonstrating AKS setups and advanced Kubernetes concepts.

## Features
- Deploy and manage Kubernetes clusters in Azure.
- Implement Horizontal Pod Autoscaling (HPA), Persistent Volumes (PV), and RBAC.
- Enable monitoring and observability with Azure Monitor and Application Insights.

## Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/aks-lab.git
   cd aks-lab
   ```
2. Run the setup script to initialize the lab:
   ```bash
   ./scripts/init.sh
   ```

## Cleanup
To delete all resources, run:
```bash
./scripts/cleanup.sh
```
