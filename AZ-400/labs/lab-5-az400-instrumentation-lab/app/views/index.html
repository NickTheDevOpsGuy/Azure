👋 Welcome to the AZ-400 Instrumentation Lab

This app demonstrates Application Insights with Node.js.

  • Check health status
  • Send custom metric
  • Trigger a fake error

🔍 Telemetry Routes

  • / → Tracks HomePageVisited event
  • /health → Sends HealthCheck metric
  • /simulate-error → Triggers Exception
  • /custom-metric → Sends CustomLatency metric

Open Azure Application Insights → Logs to query your data with KQL.
