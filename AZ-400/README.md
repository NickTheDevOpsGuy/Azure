# 🧠 AZ-400: Designing and Implementing Microsoft DevOps Solutions

Welcome to my official AZ-400 preparation hub — built in public and aligned with real-world DevOps practices.

This folder contains hands-on labs, documentation, study checklists, and notes that directly support the skills required to pass the **AZ-400: Designing and Implementing Microsoft DevOps Solutions** certification exam.

---

## 📚 What’s Inside

### 🔬 `labs/`
Realistic, exam-aligned DevOps labs with full code, YAML pipelines, and Azure integration.

- `app-insights-monitoring-lab/` – Monitor applications with Application Insights + KQL
- `lab-1-monitoring-storage/` – A sample Kubernetes project
- `lab-2-storage-private-endpoint/` – Deploys a Storage Account with a Private Endpoint and Private DNS Zone for secure access.
- `lab-3-keyvault-secert/` – Deploys an Azure Key Vault with a sample secret and access policies for secure management.
- `lab-4-terraform-rg/` – Deploys an Azure Resource Group, Storage Account, and Log Analytics Workspace using Terraform.
- `lab-5-az400-instrumentation-lab/` – Deploys an Azure Resource Group, Storage Account, and Log Analytics Workspace using Terraform.
- `lab-6-devops-project-tracking-azure-boards-github/` – Connect GitHub commits to Azure Boards work items for seamless project tracking and traceability.
- `lab-7-pipelines-variable-scope/` – Hands-on lab focused on mastering Azure DevOps Pipeline variables across multiple scopes.
- `lab-8-service-connections-managed-identities/` – Connect GitHub commits to Azure Boards work items for seamless project tracking and traceability.
- `lab-9-multi-stage-deployment-gates//` – This lab builds a full production-style Azure DevOps multi-stage pipeline with real gated deployments across Dev, Staging, and Production environments.
